# Smart Password Manager (Python)

A simple and secure command-line password manager built using Python.  
This project demonstrates core Python concepts such as file handling, JSON storage, modular code structure, and user interaction.

## Features
- Add and store passwords securely
- Retrieve saved credentials by website name
- Persistent storage using JSON
- Simple and beginner-friendly CLI interface

## Tech Stack
- Python 3.12
- JSON
- File Handling
- Command Line Interface (CLI)
